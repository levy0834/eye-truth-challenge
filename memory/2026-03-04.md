### 4. A 股量化数据源与 T+1 交易调研
**核心发现**：
- ❌ A股无真正实时数据（免费源延迟 3-10分钟）
- ✅ 日线/分钟线策略可行（隔日交易）
- ⚠️ T+1 制度：收盘后算信号，次日开盘执行
- 推荐策略：MA均线、布林带、动量轮动（510300.SH）

**数据源评估**：
- AKShare：免费但延迟大、稳定性差（爬虫易失效）
- Tushare Pro：分钟级，需 token（有免费额度）
- yfinance：美股为主，延迟15分钟

**结论**：MVP 可实现低频隔日交易（日线级别）

**报告文件**：`reports/a-share-quant-research_2026-03-04.md`

### 5. A股ETF量化研究系统开发
创建项目 `projects/a-share-etf-quant/`：

**文件结构**：
```
├── README.md                  # 项目说明
├── run_all.sh                 # 一键启动脚本
├── scripts/
│   ├── fetch_etf_data.py      # 数据抓取（10年历史）
│   ├── explore_strategies.py  # 策略回测与评估
│   └── fetch_backup_yfinance.py # 备用方案（美股ETF）
└── data/
    ├── raw/                   # 原始数据
    ├── cache/                 # 缓存
    └── processed/             # 处理后的特征
```

**技术特性**：
- 双重数据源：优先 akshare（A股ETF），备用 yfinance（美股ETF如 SPY）
- 回测引擎：支持买入/卖出/持仓，自动风险控制（仓位≤10%，止损-3%，止盈+8%）
- 5种内置策略：MA Cross, MACD Cross, RSI Extreme, Bollinger Band, TAV Confluence
- 自动划分训练集(2015-2021)和测试集(2022-2025)
- 输出：策略对比表、最优策略、资金曲线图
