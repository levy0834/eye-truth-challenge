## 🔧 技术细节

### Bug修复详情

**问题代码**:
```python
# 原代码（错误）
if side == 'sell':
    pos = self.positions.get(symbol)
    if pos:
        pnl = (fill_price - pos.avg_price) * quantity
    # 之后删除pos...
```

**修复后**:
```python
if side == 'sell':
    pos = self.positions[symbol]
    avg_price = pos.avg_price  # 先保存
    pos.quantity -= quantity
    if pos.quantity == 0:
        del self.positions[symbol]
    pnl = (fill_price - avg_price) * quantity  # 使用保存的值
```
