# AGENTS.md - 项目元信息

- 项目名称：eye-truth-challenge
- 别名：眼力试炼、真假识破
- 类型：微信 H5 小游戏原型
- 目标用户：喜欢挑战、眼力测试、快速反应的玩家
- 核心玩法：有限资源 + 视觉判断 + 时机压力
- 开发时间：2026-03-11 小时级原型
- 状态：MVP 完成，可部署

## 运行要求
- 现代浏览器（支持 ES6 + Canvas）
- 移动端优先，也支持桌面端

## 维护说明
- 游戏逻辑在 `game.js` 中
- 样式在 `style.css` 中
- 如需新增图形类型，扩展 SHAPES 数组
- 如需调整难度，修改 GAME_CONFIG.rounds
